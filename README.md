# MBNECOM LLC - AI Automation & Business Solutions

Professional website for **MBNECOM LLC** (MBN LLC), an AI automation and business solutions company led by **Muhammad Muhammad**.

## Tech Stack

- [Next.js 16](https://nextjs.org/) — React framework
- [TypeScript](https://www.typescriptlang.org/) — Type safety
- [Tailwind CSS 4](https://tailwindcss.com/) — Utility-first styling

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

## Pages

- **Home** (`/`) — Hero, services overview, company stats, CTA
- **About** (`/about`) — CEO profile, company values, timeline
- **Services** (`/services`) — AI Automation, App Development, BI, API Integrations, Consulting, Infrastructure
- **Contact** (`/contact`) — Contact form, company info, office hours

## Scripts

| Command | Description |
|---------|-------------|
| `npm run dev` | Start development server |
| `npm run build` | Create production build |
| `npm run start` | Start production server |
| `npm run lint` | Run ESLint |
